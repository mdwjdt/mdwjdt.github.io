属天的奥秘 第7447节

7447.“耶和华就这样行”表照所说的而成就。这是显而易见的，无需解释。

Potts(1905-1910) 7447

7447. And Jehovah did so. That this signifies that, as said, so it was done, is evident without explication.

Elliott(1983-1999) 7447

7447. 'And Jehovah did so' means as it had been declared, so it was done. This is clear without explanation.

Latin(1748-1756) 7447

7447. `Et fecit Jehovah ita': quod significet ut dictum ita factum, constat absque explicatione.
